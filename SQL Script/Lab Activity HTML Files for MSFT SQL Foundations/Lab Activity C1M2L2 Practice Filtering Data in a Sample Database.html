Practice Filtering Data in a Sample Database

This hands-on activity strengthens your skills in filtering data using the SQL WHERE clause in a practical environment. By engaging with the Chinook Database, you will practice constructing queries that apply conditional logic to retrieve specified data sets.

Learning Outcomes

  • Filter data using the WHERE clause in SQL.
  • Extract specific data using comparison operators.
  • Use logical operators to refine query outputs.

Getting Started

Open Visual Studio Code (VS Code) and connect to the Chinook database using the SQL Server extension. Right-click on the Chinook connection to create a new query.

Tasks

Task 1: Retrieve the first and last name of customers located in Berlin

Objective: Use the WHERE clause to filter records based on specific criteria.

Steps:

  • Write a query using the WHERE clause to filter customers who live in Berlin.

-- Task 1 Solution
SELECT FirstName, LastName
FROM Customer
WHERE City = 'Berlin';

Task 2: Extract tracks which are more expensive than $0.99

Objective: Use comparison operators to filter records based on specific criteria.

Steps:

  • Write a query using the WHERE clause to extract all the tracks from the Track table with a UnitPrice greater than $0.99.

Tip: The greater-than symbol (>) will result in all tracks costing $1 or more. To also include tracks costing $0.99, use the greater-than-or-equal-to symbol (>=).


-- Task 2 Solution
SELECT Name, UnitPrice
FROM Track
WHERE UnitPrice > 0.99;

Task 3: Identify customers who are located in either Paris or Rome

Objective: Practice combining conditions with logical operators.

Steps:

  • Formulate a query using the WHERE clause and the OR operator to encompass both cities, Paris or Rome.

-- Task 3 Solution
SELECT FirstName, LastName, Country
FROM Customer
WHERE City = 'Paris' OR City = 'Rome';

Task 4: Find all the invoice details for customers in Toronto with an invoice total between $5 and $10

Objective: Integrate advanced filtering techniques to retrieve specific data subsets.

Steps:

  • Use the BETWEEN and AND operators to create combined conditions for data output.

-- Task 4 Solution
SELECT *
FROM Invoice
WHERE Total BETWEEN 5 AND 10
AND BillingCity = 'Toronto';

Summary

In this activity, you explored more advanced filtering skills by combining operators with the WHERE clause, enabling you to retrieve specific data subsets. Through practice, reflection, and feedback, you will strengthen your command over basic querying techniques essential for effective data manipulation.

Key Points

SQL queries can result in refined datasets when filtered using operators, resulting in more useful data outputs.
