Retrieve Data from a Database Using SELECT

In this activity, you will put into practice the core SQL skills acquired in the lesson by crafting and executing SELECT statements. This hands-on exercise allows you to experience writing queries that retrieve data from the Chinook Database, applying the concepts taught in the module.

Learning Outcomes

  • Write SQL SELECT statements to retrieve specific data from a database.
  • Use the WHERE clause to filter records.
  • Sort query results using the ORDER BY clause.

Getting Started

Open Visual Studio Code (VS Code), and connect to the Chinook database using the SQL Server extension.

Tasks

Task 1: Retrieve the Name and Email of All Customers

Objective: Practice retrieving data from specific table columns, reinforcing basic SELECT syntax.

Steps:

  1. On the first line, type ‘SELECT’ and then specify the columns you want to retrieve each separated by a comma: FirstName, LastName, Email.
  2. On the second line, type ‘FROM’ and then specify the table from which to retrieve the data: Customer.

Tip: Don’t forget to end your code with a semicolon ( ; ).

Task 2: Extract Information for Customers Located in Germany

Objective: Use the WHERE clause to filter records based on specific criteria.

Steps:

  1. On the first line, type ‘SELECT’ and then specify the columns you want to retrieve: FirstName, LastName, and City.
  2. On the second line, type ‘FROM’ and then specify the table from which to retrieve the data: Customer.
  3. On the third line, type ‘WHERE’ and then specify the criteria by which to filter: Germany.

Tip: Ensure there are no whitespaces inside the quotation marks indicating the location.

Task 3: Get a List of All Song Tracks Sorted by Their Name in Ascending Order

Objective: Practice using the ORDER BY clause for sorting query results.

Steps:

  1. On the first line, type ‘SELECT’ and then specify the columns you want to retrieve.
  2. On the second line, type ‘FROM’ and then specify the table from which to retrieve the data.
  3. On the third line, type ‘ORDER BY’ and specify which column to sort and whether to sort by ascending (ASC) or descending (DESC) order.

Task 4: Identify the Rock Genre Tracks and Present Them Sorted in Ascending Order by Name

Objective: Integrate filtering and sorting for more effective data retrieval.

Steps:

  1. On the first line, type ‘SELECT’ and then specify the columns you want to retrieve.
  2. On the second line, type ‘FROM’ and then specify the table from which to retrieve the data.
  3. On the third line, type ‘WHERE’ and filter for the Rock genre ID.
  4. On the fourth line, type ‘ORDER BY’ and specify which column to sort in ascending order (ASC).

Tip: The GenreId for “Rock” tracks is 1.

Summary

In this activity you explored foundational SQL skills, enabling you to confidently approach data retrieval tasks as you continue to develop your SQL expertise. Through practice, reflection, and feedback, you will strengthen your command over basic querying techniques essential for effective data manipulation.

Key Points

SQL queries are built on the basic SELECT command, using other clauses to filter and sort data output.

Reference Solution

After completing this activity, take a moment to review the exemplar solution. This sample solution can offer insights into different techniques and approaches. Reflect on what you can learn from the exemplar solution to improve your coding skills.

Remember, multiple solutions can exist for some problems; the goal is to learn and grow as a programmer by exploring various approaches. Use the exemplar solution as a learning tool to enhance your understanding and refine your approach to coding challenges.

Exemplar Solutions:

-- Task 1: Retrieve the name and email of all customers
SELECT FirstName, LastName, Email
FROM Customer;

-- Task 2: Extract information for customers located in Germany
SELECT FirstName, LastName, City
FROM Customer
WHERE Country = 'Germany';

-- Task 3: Get a list of all song tracks sorted by their Name in ascending order
SELECT Name
FROM Track
ORDER BY Name ASC;

-- Task 4: Identify the rock genre tracks and present them sorted in ascending order by name
SELECT Name
FROM Track
WHERE GenreId = 1 -- GenreId for Rock is 1
ORDER BY Name ASC;
