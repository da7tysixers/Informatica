Apply SQL Functions to Transform Data

Overview

This hands-on activity is designed to help you harness the power of SQL functions to transform data in the Chinook Database. By applying functions such as string manipulation, mathematical calculations, and date transformations, you will develop the ability to modify and prepare raw data for meaningful analysis.

Learning Outcomes

By the end of this activity, you will be able to:

  • Transform and optimize data using SQL functions.
  • Standardize data using string functions.
  • Use mathematical functions to automate numeric calculations.
  • Enhance readability of data with date functions.
  • Use aggregation functions to summarize data.

Getting Started

Open VS Code, and connect to the Chinook database using the SQL Server. Right click on the Chinook connection to create a new query.

Task 1: String Functions: Standardize Artists’ Names

Objective: Utilize string functions for data standardization, an essential step for uniform data presentation.

Steps:

  1. Use the following starter code and write a query to retrieve and transform all artist names to uppercase. 
    -- Retrieve and transform all artists' names to uppercase. 
SELECT -- Your code here.
FROM -- Your code here.
  2. Tip: Remember that functions are nested within the SELECT statement prior to defining FROM, filtering (WHERE), or ordering (ORDER BY).

Task 1 Solution:

FROM Artist;

Task 2: Mathematical Functions: Calculate Average Track Length

Objective: Use mathematical functions to perform numeric calculations and gain insights from data.

Steps:

  1. Use the following starter code and write a query using mathematical functions to derive the average track length in minutes rounded to two decimal places. 
    -- Calculate the average track length in minutes from the Track table.
SELECT -- Your code here.
FROM -- Your code here.
  2. Tip: Track length is recorded within the table as milliseconds. There are 60,000 milliseconds in 1 minute.

Task 2 Solution:

FROM Track;
