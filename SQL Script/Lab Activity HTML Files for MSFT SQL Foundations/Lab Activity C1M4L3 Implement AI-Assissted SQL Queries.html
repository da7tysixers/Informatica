Implement AI-assisted SQL Queries

Overview

In this activity, you will engage in a hands-on experience using GitHub Copilot by Microsoft to create and optimize SQL queries. This interactive exercise is designed to demonstrate how generative AI can assist in making SQL query writing more efficient and effective. By comparing AI-generated queries to manually written ones, you will gain insight into the capabilities and efficiency of using AI tools in SQL database management.

Getting Started

Open VS Code, and connect to the Chinook database using the SQL Server. Ensure you’re signed into GitHub and the Copilot integration is active. Right click on the Chinook connection to create a new query.

Tasks

Task 1: Basic Query Generation with Copilot

Objective: Formulate a query retrieving track names for all tracks in the Track table where the genre is ‘Rock’.

Steps:

  1. Start with a SELECT statement and observe Copilot’s suggestions for completing your query.
  2. Compare the AI-generated query with your manually written attempt.
  3. Tip: Pay attention to how Copilot suggests completing the query.
  4. Test Your Work: Check if the query returns the expected track names for the ‘Rock’ genre.

Task 1 Solution:

SELECT Name
FROM Track
WHERE GenreId = (
    SELECT GenreId
    FROM Genre
    WHERE Name = 'Rock'
);

SELECT
    c.FirstName AS CustomerFirstName,
    c.LastName AS CustomerLastName,
    i.InvoiceId,
    i.InvoiceDate,
    i.Total
FROM
    Customer c
JOIN
    Invoice i ON c.CustomerId = i.CustomerId
WHERE i.Total > 10 
ORDER BY i.InvoiceDate DESC;

Task 3 Solution:

SELECT
    ar.Name AS ArtistName,
    al.Title AS AlbumTitle,
    SUM(il.UnitPrice * il.Quantity) AS TotalSales
FROM Artist ar
JOIN Album al ON ar.ArtistId = al.ArtistId
JOIN Track t ON al.AlbumId = t.AlbumId
JOIN InvoiceLine il ON t.TrackId = il.TrackId
GROUP BY ar.Name, al.Title
ORDER BY TotalSales DESC;

Common Issues & Solutions

  • Problem: Ambiguous column references in joins.
    Solution: Ensure proper use of table aliases and qualify column names.
  • Problem: AI suggestions do not meet requirements.
    Solution: Double-check and refine the query, combining manual adjustments with AI suggestions.

Summary

Leveraging AI tools like GitHub Copilot by Microsoft can greatly enhance the efficiency and accuracy of SQL query writing. While AI can assist in generating and optimizing queries, it is important to validate and modify AI-generated code to fit specific needs accurately.

Key Points

  • AI tools can streamline the generation of SQL queries, saving time and effort.
  • Manual validation and adjustments ensure the accuracy and relevance of AI suggestions.
  • Reflecting on the use of AI can provide insights into optimizing SQL query management.
