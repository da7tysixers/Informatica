Accessing GitHub Copilot by Microsoft

Overview

In this activity, you will connect GitHub Copilot to the Chinook database in your VS Code environment.

Getting Started

Open VS Code, and follow the instructions to set up GitHub Copilot.

Tasks

Task 1: Connect GitHub Copilot

Steps:

  1. Open VS Code, then press "Ctrl + Shift + P" to open the command palette. Type "GitHub: Sign in" and press "Enter" to initiate the login process. Alternatively, you can open the side activity bar using the keyboard shortcut "Ctrl + B" to access the "Accounts" icon. Select "Sign in with GitHub" from there.
  2. Follow the prompts to sign into your GitHub account by entering your username and password.
  3. Verify your identity with multi-factor authentication if prompted for additional security.
  4. If an authorization prompt appears, you may need to grant access each time you log in. When the authorization window opens, navigate using your screen reader to the "Always allow vscode.dev to open links" checkbox, then tab to the "Open xdg-open" button and press "Enter".

Task 2: Confirm the Connection of GitHub Copilot with the Chinook database

Steps:

  1. Connect to the Chinook database in VS Code. Navigate to the bottom bar using the screen reader shortcuts and select "Connections". Choose the Chinook database from the list and select "New Query".
  2. Begin typing a standard SELECT statement in the new query window: SELECT * FROM.
  3. As you type, observe suggestions from GitHub Copilot, which will assist you in completing the query. For example, you will see suggestions for table names contained in the Chinook database such as "Customer" or "Invoice".
  4. Select the desired table name, such as "Customer", and continue typing. GitHub Copilot will provide suggestions for the next parts of the query.
  5. To filter customers by country and city, type the following statement, then press "Tab" to accept the auto-completion provided by Copilot:
SELECT *
FROM Customer
WHERE Country = 'USA' AND City = 'Redmond';

Common Issues & Solutions

  • Problem: I don’t have a GitHub account. How do I get one?
    Solution: Go to GitHub and sign up. You can follow these steps to set up an account.
  • Problem: I’m using a Mac and can’t click on the Chinook database to get “New Query”.
    Solution: Use two fingers, tap on the touchpad, and select “New Query”.
  • Problem: I don’t see any suggestions from Copilot as I type.
    Solution: Ensure you have created a GitHub account and are signed into VS Code.
